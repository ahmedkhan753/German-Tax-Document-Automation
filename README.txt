README: Document Processor Automation Tool

**Description**: This Python tool (as .exe) automates merging tax docs, converting Word to PDF, and adding watermarks per your spec.

**Requirements**:
- Windows PC (with MS Word installed for docx2pdf).
- Folders: Create 'input/' (put client docs here), 'watermarks/' (put your 3 Wasserzeichen PDFs), 'output/' (final PDF here).

**How to Run**:
1. Copy docs to 'input/Daten Franklin/' (or modify CONFIG in script if needed).
2. Double-click document_processor.exe.
3. Check 'output/final_output.pdf' â€“ merged with watermarks.

**Troubleshooting**:
- If no MS Word: MS Word is required for .docx conversion.
- Logs: Check the console window for any error messages.
- Files: Ensure watermark files are named correctly:
  - 'Wasserzeichen Anschreiben.pdf'
  - 'Wasserzeichen Deckblatt.pdf'
  - 'Wasserzeichen Allgemein.pdf'
